.config/nvim
==============

Tested with: 
- neovim 0.9.5
- go 1.22
- node 1.12
- npm 8.5.1
- Ubuntu 22.04 LTS

Uses: 
- lazy (nvim plugin manager)
- mason (nvim package manager for language tools)

Other OS Requirments:
- gcc, make, cmake
- ripgrep

Based on:
https://www.josean.com/posts/how-to-setup-neovim-2024
